# Music_Generator
A Python project of course Data Mining and Algorithm of Machine Learning
Author:Yuan Qu
Date: 2017.02.27

##Steps

1. 读取音乐文件  .mp3 --> .wav --> 波形序列
2. 波形特征提取
3. k-means 聚类
4. 可视化
5. 学习

##File Organization

1. Generate: 用于测试音乐生成功能
2. k-means: 用于测试 k-means 聚类方法
3. mp3: 存放 mp3 文件
4. wav: 转换输出后的 wav 文件
5. data: 波形提取文件和处理后的特征文件
